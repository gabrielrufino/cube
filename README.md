# Cube

Data Structures and Algorithms made in Typescript

### Getting started

##### Installing package

```bash
npm i @gabrielrufino/cube
```

##### Importing package

<small>ESModules:</small>
```js
import cube from '@gabrielrufino/cube'
```

<small>CommonJS:</small>
```js
const cube = require('@gabrielrufino/cube')
```
